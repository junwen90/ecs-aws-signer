# Break-Glass Incident Register

## Purpose
Track all break-glass exceptions for container image signing bypass.

## Format

| Date | Time | Requestor | Reason | Image Digest | Approved By | Deployed | Restored | Notes |
|------|------|-----------|--------|--------------|-------------|----------|----------|-------|
|      |      |           |        |              |             |          |          |       |

## Procedure

1. Fill in all fields at time of incident
2. DevOps lead approval required before deployment
3. Restore normal signing after deployment
4. Review within 48 hours
5. Archive after 90 days
